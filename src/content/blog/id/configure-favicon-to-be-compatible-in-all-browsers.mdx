### Apa itu Favicon?

Favicon, singkatan dari "favorite icon," adalah ikon kecil yang muncul di sebelah judul halaman web pada tab browser. Bahkan kita menambahkan website PWA favicon ini digunakan untuk tampilan icon di mobile atau desktop.

![contoh favicon ruang work](@/assets/blog/configure-favicon-to-be-compatible-in-all-browsers/contoh-favicon-ruang-work.png)

contohnya adalah logo RW pada tab broser di atas.

### Fungsi Favicon

-   **Identitas Situs Web**

Seperti tanda merek atau tanda pengenal di situs website. gunanya sebenernya cuman ngebedain dari website lain. semisal config tidak di config jadinya seperti,

![halaman tanpa configurasi favicon](@/assets/blog/configure-favicon-to-be-compatible-in-all-browsers/site-tanpa-configurasi-favicon.png)

-   **Identitas Visual**

intinya favicon menjadi salah satu identitas visual wajib. seperti ketika situs kita di buka, di tambahkan di home screen atau situs kita di bookmark. favicon ini bisa menjadi salah satu cara untuk branding dan menambah kredibilitas agar situs kita terlihat profesional.
